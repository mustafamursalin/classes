Q1. Create two tables:
Manufacturer (
    id (auto increment), 
    name (varchar(50)), 
    address (varchar(100)), 
    contact_no (varchar(50))
    );

Product (
    id (auto increment), 
    name (varchar(50)), 
    price (INT(5)), 
    manufacturer_id (INT(10))
    );

Q2. Create a form and write PHP script to insert data into Manufacturer table using stored procedure.

Q3. Create an after delete trigger which will delete record(s) from the product table when any corresponding manufacturer_id are deleted from the manufacturer table.

Q3. Create a view to display all records from the product table where price is greater than 5000. Write PHP script to display the data from the view.
